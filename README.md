### Almost Simple Metro Simulator (ASMS)
Це відносно простий і дуже сирий симулятор метро.

* **Мова:** C++
* **Бібліотеки:** SFML, TGUI, EnTT

### Плани на майбутнє:
- [x] Перенести консоль в середину вікна гри
- [x] Розмістити всі спрайти вагону на ньому
- [ ] Зробити варіант вагону в другу сторону
- [ ] Повноцінна карта
- [ ] Пасажири
- [ ] Доробити логіку вагону типу Е
- [ ] Більше вагонів з сім'ї типів Е
- [ ] Система АРШ/АЛС
- [ ] Система екранів, меню, налаштування, збирання складу
- [ ] Можливість ходити по поїзду, вийти на станції


## Як зібрати проект

Проект максимально підготовлений до роботи. Усі залежності вже включені в репозиторій, тому додаткове завантаження бібліотек не потрібне.

### Покрокова інструкція:
1. Склонуйте репозиторій або завантажте його як ZIP-архів
2. Перейдіть до папки проекту та запустіть файл **`AlmostSimpleMetroSim.sln`** за допомогою Visual Studio 26
3. Переконайтеся що зверху вибрана конфігурація **`Release`**, Платформа повинна бути встановлена **`x64`**
4. Натисніть клавішу **`F5`** або кнопку **"Local Windows Debugger"**

Дод. інформація: У Visual Studio має бути встановлено **"Desktop development with C++"**
