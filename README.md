### Almost Simple Metro Simulator (ASMS)
Це відносно простий і дуже сирий симулятор метро.

* **Мова:** C++
* **Бібліотеки:** SFML, TGUI, EnTT

### Плани на майбутнє:
- [x] Перенести консоль в середину вікна гри
- [x] Розмістити всі спрайти вагону на ньому
- [ ] Зробити варіант вагону в другу сторону
- [ ] Повноцінна карта
- [ ] Пасажири
- [ ] Доробити логіку вагону типу Е
- [ ] Більше вагонів з сім'ї типів Е
- [ ] Система АРШ/АЛС
- [ ] Система екранів, меню, налаштування, збирання складу
- [ ] Можливість ходити по поїзду, вийти на станції
