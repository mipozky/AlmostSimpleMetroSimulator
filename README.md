# AlmostSimpleMetroSimulator
Almost simple metro simulator(ASMS) це відносно простий і дуже сирий симулятор метро

Мова: С++
Основні бібліотеки: SFML, TGUI, Entt

Плани на майбутнє:
[x] перенести консоль в середину вікна гри
[ ] розмістити всі спрайти вагону на ньому
[ ] зробити варіант вагону в другу сторону
[ ] повноцінна карта
[ ] пасажири
[ ] Доробити логіку вагону типа Е
[ ] більше вагонів з сім'ї типів Е
[ ] Система АРШ/АЛС
[ ] система екранів, меню, налаштування, збирання складу
[ ] можливість ходити по поїзду, вийти на станції
